# Kit de Tareas Recurrentes para Hostelería — Product Plan

## Product Definition
- **Name**: Kit de Tareas Recurrentes Pro
- **Slug**: `kit-tareas`
- **Format**: kit (Excel templates — multiple workbooks)
- **Price**: €19 (launch) / ~~€49~~ (strikethrough)
- **Audience**: Gerentes, encargados, jefes de cocina, propietarios de restaurantes, cafeterías, pastelerías, hoteles
- **Problem**: Los restaurantes funcionan con cientos de tareas que se repiten cada día, cada semana, cada mes. Sin checklists escritos, las tareas se olvidan, la calidad baja, y cuando cambia el personal (rotación del 30-40%) se pierde el know-how. Trail cobra €60-75/mes por sitio. Este kit lo resuelve con Excel por €19, pago único.
- **Tagline**: "Todas las tareas de tu restaurante, organizadas por turno, área y perfil — listas para imprimir"
- **Referencia competitiva**: Trail (trailapp.com) — £60-75/mes por local. Nosotros: €19 pago único.

## Deliverables Structure

## Excel Templates (8 workbooks)

### Workbook 1: Tareas de Apertura y Cierre
**Pestañas:**
- Apertura Cocina (mise en place, encender equipos, recibir mercancía, briefing)
- Apertura Sala (montar mesas, revisar reservas, preparar estaciones, terraza)
- Apertura Barra (stock, cortes de fruta, máquina café, hielo)
- Cierre Cocina (limpiar partidas, guardar producto, apagar equipos, FIFO)
- Cierre Sala (recoger, limpiar, reponer, cuadrar caja)
- Cierre Barra (inventario rápido, limpiar cristalería, cerrar cámaras)

Cada tarea: ☐ checkbox + responsable + hora límite + notas

### Workbook 2: Tareas por Partida de Cocina
**Pestañas:**
- Partida de Calientes (plancha, fogones, horno, freidora)
- Partida de Fríos (ensaladas, carpaccios, ceviches, plato frío)
- Partida de Pastelería (postres, bollería, pre-elaboraciones)
- Partida de Cuarto Frío (mise en place, cortes, marinados)
- Mise en Place General (checklist diario de prep)

Cada pestaña: tareas antes del servicio, durante el servicio, después del servicio

### Workbook 3: Tareas del Manager / Gerente
**Pestañas:**
- Checklist Diario del Manager (revisar reservas, briefing equipo, comprobar plantilla, revisar incidencias, caja, RRSS)
- Checklist Semanal (reunión equipo, inventario, pedidos, rotación FIFO, revisión proveedores, planificación carta)
- Checklist Mensual (P&L, evaluación personal, mantenimiento equipos, marketing, formación)
- Control de Reservas (seguimiento diario de reservas, no-shows, ocupación)
- Handover / Cambio de Turno (checklist de traspaso entre turnos)

### Workbook 4: Tareas por Perfil Profesional
**Pestañas:**
- Chef Ejecutivo (tareas diarias/semanales/mensuales)
- Sous Chef (tareas diarias)
- Jefe de Sala / Maître (tareas diarias)
- Barman (tareas diarias de barra)
- Runner / Auxiliar (tareas por turno)
- Personal de Limpieza (tareas por zona)

### Workbook 5: Tareas Semanales y Mensuales
**Pestañas:**
- Limpieza Profunda Semanal (por zona: campana, cámaras, horno, freidora, suelos, cristales)
- Limpieza Profunda Mensual (conductos, desagües, estanterías, techos)
- Revisión FIFO Semanal (cámaras, almacén, congeladores)
- Inventario Semanal (conteo rápido por categoría)
- Mantenimiento Equipos Mensual (plancha, horno, lavavajillas, campana, cámaras)
- Control de Caducidades (revisión semanal por zona)

### Workbook 6: Tareas para Eventos y Festivos
**Pestañas:**
- Checklist Pre-Evento (48h antes, 24h antes, día del evento)
- Checklist Día del Evento (montaje, servicio, desmontaje)
- San Valentín (tareas específicas de decoración, menú especial, turnos reforzados)
- Nochevieja / Navidad (prep especial, stock extra, turnos, menú)
- Terraza Temporada (montaje inicio temporada, desmontaje fin)
- Evento de Catering Externo (transporte, montaje, servicio, recogida)

### Workbook 7: Tareas por Tipo de Negocio
**Pestañas:**
- Restaurante Casual (apertura/cierre adaptado)
- Cafetería / Brunch (tareas específicas: cafetera, vitrina, bollería)
- Pastelería / Obrador (tareas producción: masas, horneado, decoración, vitrina)
- Dark Kitchen (tareas delivery: tablets, packaging, riders, stock envases)
- Buffet Hotel (montaje buffet, reposición, control temperaturas, recogida)
- Food Truck (apertura móvil, generador, stock limitado, cierre)

### Workbook 8: Plantilla Maestra Personalizable
**Pestañas:**
- Plantilla en Blanco por Franja Horaria (para que el usuario cree sus propias tareas)
- Plantilla en Blanco por Área
- Plantilla en Blanco por Perfil
- Instrucciones de Personalización

### BONUS
- **Bonus 1**: Plantilla de Briefing de Servicio — Checklist para la reunión pre-servicio con el equipo (platos del día, alérgenos, VIPs, incidencias) — valor €7
- **Bonus 2**: Calendario Anual de Tareas Especiales — Excel con los 12 meses y las fechas clave de hostelería (San Valentín, Semana Santa, Navidad, festivos locales) con tareas asociadas — valor €9

## Format & UX of Each Template

Cada Excel sigue este formato:

```
| ☐ | Tarea | Área | Responsable | Hora Límite | Completada | Firma |
|---|-------|------|-------------|-------------|------------|-------|
```

- Checkbox ☐ para marcar como completada
- Colores por zona: Cocina (verde), Sala (azul), Barra (naranja), Limpieza (morado)
- Validación de datos para responsable (desplegable con puestos)
- Formato optimizado para imprimir en A4
- Agrupadas por franja horaria con separadores visuales
- Espacio para firma del encargado al final de cada turno
